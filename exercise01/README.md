# Exercise 1

Basics:
```
./gradlew tasks --all

./gradlew foo

./gradlew bar

./gradlew foo bar
```

Task Ordering:
```
./gradlew foobar
```

Default Task:
```
./gradlew
```

Continue with failed tasks:
```
./gradlew uno dos

./gradlew --continue uno dos
```
